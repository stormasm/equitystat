Zrato Quick Tutorial

Last modified on Friday June 24, 2005

Before running the programs in this tutorial be sure you have read the installation document. Assuming Zrato is properly installed and the simple test at the end of the installation document has run successfully, then you are now ready to run the quick tutorial.

The purpose of running the quick tutorial is to further confirm that your system is properly installed and working. Once, you have gone thru the steps below, you should then followup up with the Tutorial so you can understand what all of the steps actually did.

Tutorial Step One, Populate the Ma (Moving Average) Table in the Database

  • ant ma

Tutorial Step Two, Generate the Moving Average Report

  • ant gm

Tutorial Step Three, Generate the Technology Sector Report

  • ant gs

Tutorial Step Four, Generate the EPS (Earnings Per Share) Report

  • ant ge

Tutorial Step Five

For future reference, if you want to generate all of the reports at once instead of going through the individual steps above simply type.

  • ant ga

Zrato Installation Doc for Windows

Last modified on Friday June 24, 2005

Installation Step One, Download and Install Packages from the internet

  • Download and install Java 2 Platform Standard Edition 5.0 from Sun Microsystems
  • Download and install Apache Ant 1.6.5 from Apache
  • Download and install the MySql Database Server 4.1 from MySql
  • Download and install Hibernate from Hibernate

Installation Step Two, Install the Zrato Database.

  • Shut down your MySql database server
  • Unzip the zrato database and place it in the data directory wherever MySql is installed.
  • Also, in that directory you will find a directory called mysql and test.
  • With MySql each directory is physically a database, so by adding zrato you are installing another new database.
  • The zrato database is a directory of files with names in it like hp.*, and ks.*, etc...
  • Each set of three files is a table in the database.
  • Start up your MySql database server. The zrato database should now be loaded.
  • You may want to bring up a MySql GUI to review the database tables in the Zrato database.

Installation Step Three, Create an environment variable.

  • Create an environment variable called ETSDATA_HOME=c:\etsdata
  • If you don't know how to do this here are the instructions.
  • Bring up the control panel
  • Double click on System
  • A System properties window should now appear.
  • Double click on the Advanced Tab
  • At the bottom is a button called Environment Variables, click on that.
  • Under the User Variables click on New
  • For Variable Name put in ETSDATA_HOME
  • For Variable Value put in c:\etsdata
  • Click on OK several times to instantiate.
  • In a new command window at the c:\ prompt type set
  • And make sure the above environment variable has been set

Installation Step 4A, Create the Ant environment variable.

  • If Ant was already installed on your machine, you may have this environment variable already set.
  • If not, then set it using the same procedure as above.
  • Call the environment variable ANT_HOME
  • Set the environment variable to the location where ANT is installed.

Installation Step 4B, Create the Hibernate environment variable.

  • If Hibernate was already installed on your machine, you may have this environment variable already set.
  • If not, then set it using the same procedure as above.
  • Call the environment variable HIB_HOME
  • Set the environment variable to the location where Hibernate is installed.

Installation Step Five, Install Zrato

  • Unzip the installation and place under c:\etsag

Installation Step Six, Test to see if configured properly

This will test to make sure that these software packages are all installed correctly.

  • Ant
  • Java
  • MySql
  • Hibernate
  • Zrato

Run the following test in c:\etsag to make sure your Zrato system is correctly installed

  • ant ic
